  • Determine and set margin boxes' sizes along variable dimension using an algorithm specified in CSS Paged Media spec.

    Parameters

          [key: string]: Container;
      }

      Containers corresponding to the target margin boxes in one page edge (top, bottom, left, right)

      • [key: string]: Container
    • isHorizontal: boolean

    • dimensions: {
          end: Val;
          extent: Val;

      Page dimensions. start: margin-left or margin-top. end: margin-right or margin-bottom. extent: border-box width or height of the page area (= available width or height for the target margin boxes)
